# Fifth Column Chess

Vite + React app ready for GitHub + Vercel.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy with Vercel

Push this folder to GitHub, then import the repo in Vercel.

Framework preset: **Vite**

Build command:

```bash
npm run build
```

Output directory:

```bash
dist
```
